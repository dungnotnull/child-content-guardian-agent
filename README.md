**AI-Powered Local Content Filtering & Monitoring for Child Safety.**

Child Content Guardian is a privacy-first, local-inference system designed to protect children from harmful online content. Unlike cloud-based filters, this system processes everything on your local machine, ensuring that your child's browsing history never leaves your home.

## Key Features

### AI-Driven Filtering
- **Toxicity Detection**: Real-time analysis of text for hate speech and toxicity using RoBERTa.
- **Visual Safety**: Multi-stage image analysis to detect NSFW and violent content.
- **Brain-Rot Scoring**: Heuristic analysis of video \"cut-rates\" to identify high-stimulation/low-value content.
- **Scam Detection**: Specialized detection for child-targeted gaming and gift-card scams.

### Parental Control Suite
- **Age-Band Thresholds**: Dynamic sensitivity settings for *Early Childhood*, *Middle Childhood*, and *Adolescents*.
- **Allowlist Management**: Easy domain and pattern-based exceptions.
- **AI-Powered Reports**: Weekly summaries generated by LLMs (Ollama/Claude/OpenAI) explaining why content was blocked.

### Privacy & Security
- **Local-Only**: Zero external calls for content analysis.
- **Encrypted Storage**: Local SQLite database for event logging.
- **Air-Gapped Design**: Designed to work without constant internet dependency for the core filter.

## Quick Start

### 1. Prerequisites
- Python 3.11+
- [uv](https://github.com/astral-sh/uv) (recommended for dependency management)
- [Ollama](https://ollama.ai/) (if using local LLM)

### 2. Installation
`ash
# Clone the repository
git clone https://github.com/dungnotnull/child-content-guardian-agent.git
cd child-content-guardian-agent

# Install dependencies using uv
uv sync
`

### 3. Configuration
`ash
cp .env.example .env
# Edit .env to set your API keys and preferred LLM provider
`

### 4. Running the Server
`ash
uv run python src/api/main.py
`

### 5. Load Extension
1. Open Chrome and go to chrome://extensions.
2. Enable **Developer Mode**.
3. Click **Load unpacked** and select the /extension folder in this repo.

## Architecture

- **Backend**: FastAPI + SQLAlchemy (Async)
- **ML Engine**: PyTorch + Transformers + ONNX
- **Frontend**: React (Dashboard) + JavaScript (Chrome Extension)
- **LLM Integration**: Pluggable architecture (Ollama <-> Claude <-> OpenAI)
